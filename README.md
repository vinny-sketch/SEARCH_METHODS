 Search and Optimization Pathfinding
File:search_methods.py`
Description:A Python script detailing two foundational AI pathfinding algorithms to resolve traversal routes over a custom graph network from an initial starting node to a predefined goal terminal state.
Algorithms Implemented:
  (i) Breadth-First Search (BFS):Uses a FIFO queue structure to traverse the network layer by layer, ensuring the shortest topological path is found.
  (ii) Depth-First Search (DFS):Uses a LIFO stack structure to scan deep down individual network branches before backtracking.
